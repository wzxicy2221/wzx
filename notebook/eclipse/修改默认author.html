1. 在eclipse.ini中添加
-vmargs
-Duser.name={author name}
记得一定要在-vmargs之后，否则无效。

2. 设置eclipse参数
windows-->preference 
Java-->Code Style-->Code Templates 
code-->new Java files
原来的内容是：
${filecomment} 
${package_declaration} 

${typecomment} 
${type_declaration}
添加自定义内容，添加之后内容如下：
${filecomment} 
${package_declaration} 
/** 
* @author 作者 E-mail: 
* @version 创建时间：${date} ${time} 
* 类说明 
*/ 
${typecomment} 
${type_declaration}

3. 设置eclipse参数
windows-->preference 
Java-->Code Style-->Code Templates 
Comments-->files 
按照需要修改配置，并且保证上面项的配置中有${filecomment}这项。
